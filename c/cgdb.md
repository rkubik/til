# cgdb

ncurses front-end to gdb. Similar keyboard commands to vi.

## Keyboard Shortcuts

| Key        | Description                                             |
|------------|---------------------------------------------------------|
| ESC        | switch modes                                            |
| i          | gdb mode                                                |
| j          | down line                                               |
| k          | up line                                                 |
| Ctrl + f   | page down                                               |
| Ctrl + b   | page up                                                 |
| gg         | top of file                                             |
| G          | bottom of file                                          |
| /          | search                                                  |
| n          | next forward search                                     |
| =          | grow source window 1 line                               |
| -          | shrink source window 1 line                             |
| _          | grow source window 25%                                  |
| +          | shrink source window 25%                                |
